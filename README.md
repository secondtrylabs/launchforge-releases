# LaunchForge

Game session launcher and process manager by SecondTry Labs.

## Download

Download the latest installer from the [Releases](https://github.com/secondtrylabs/launchforge-releases/releases) page.

## Features

- **Game Profiles** — Create profiles for your games with associated launch applications
- **Launch First** — Start required apps (TrackIR, etc.) before your game launches
- **Desktop Shortcuts** — One-click shortcuts that launch everything in the right order
- **Keep Alive** — Protect apps from closing on game exit (CTD protection)
- **Steam Integration** — Browse and add games from your Steam library
- **Process Monitoring** — Automatically detects running games and launches companion apps
- **Auto-Close** — Closes launched apps when your game exits
- **Custom Icons** — Set custom icons for games and applications
- **System Tray** — Runs quietly in the background
